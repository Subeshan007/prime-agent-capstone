# Enterprise Polish Design System (Linear/Notion Style)

## 1. Color System

### Primary Palette
- **Background**: `#111315` (Deep Dark)
- **Sidebar**: `#141618`
- **Surface**: `#1A1C1E` (Cards)
- **Border**: `#2A2D30` (Subtle)
- **Accent**: `#3A7BFA` (Primary Blue)

### Functional Colors
- **Text Primary**: `#FFFFFF`
- **Text Neutral**: `#D7DADC`
- **Input Bg**: `#191B1D`

## 2. Layout Grid
- **Columns**: 12
- **Gaps**: 24px (Horizontal), 32px (Vertical)
- **Padding**: 48px (Container)

## 3. Component Rules

### Cards
- **Radius**: 14px
- **Border**: 1px solid `#2A2D30`
- **Shadow**: `0 4px 12px rgba(0, 0, 0, 0.1)`
- **Padding**: 24px - 32px

### Slider (Apple Music Style)
- **Track**: `#3A7BFA` (Blue Gradient)
- **Handle**: White Circle
- **Height**: 4px

### Profile Button
- **Shape**: Rounded Rectangle (Radius 6px)
- **Content**: Icon + Initials
- **Hover**: `#2A2D30` Background

## 4. Typography
- **Font**: Inter
- **Headings**: Semi-bold, tight tracking
- **Body**: Regular, relaxed line-height
